# Калькулятор дат

Простая консольная программа, которая запрашивает две даты и показывает, сколько дней и часов осталось/прошло между ними.

## Запуск

```bash
python3 main.py
```

Поддерживаются форматы:

- `дд.мм.гггг` (например, `12.02.2021`)
- `дд <месяц> гггг` (например, `12 февраля 2021`)

## Как упаковать в `.exe`

Да, программу можно упаковать в исполняемый файл через `PyInstaller`:

```bash
python -m pip install pyinstaller
pyinstaller --onefile main.py
```

Готовый файл появится в каталоге `dist/`.
